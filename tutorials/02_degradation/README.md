# Degradation Tutorials

Advanced tutorials focused on specific degradation mechanisms including LID and LETID.

## Tutorials

1. **01_bo_lid_accelerated_test.ipynb** - Boron-Oxygen Light Induced Degradation (B-O LID) accelerated testing
2. **02_letid_accelerated_test.ipynb** - Light and Elevated Temperature Induced Degradation (LETID) accelerated testing
3. **03_letid_outdoor.ipynb** - LETID modeling for outdoor conditions
4. **04_letid_outdoor_scenario.ipynb** - LETID outdoor scenarios based on accelerated test results
5. **05_letid_passivated_wafer.ipynb** - LETID analysis for passivated wafer technologies
6. **06_vant_hoff_degradation_model.ipynb** - Van't Hoff degradation modeling approach

## Topics Covered

- Light Induced Degradation (LID)
- Light and Elevated Temperature Induced Degradation (LETID)
- Accelerated testing methods
- Outdoor degradation prediction
- Degradation kinetics
