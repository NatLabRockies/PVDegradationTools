# Geospatial Analysis

Tutorials for large-scale geospatial analysis and mapping workflows.

## Tutorials

⚠️ **All tutorials in this folder require access to High Performance Computing (HPC) resources.**

1. **01_geospatial_templates.ipynb** - Templates for geospatial analysis workflows
2. **02_geospatial_world_map.ipynb** - World-scale geospatial degradation mapping
3. **03_letid_outdoor_geospatial_demo.ipynb** - LETID outdoor geospatial demonstration
4. **04_scenario_geographical_features.ipynb** - Incorporating geographical features into scenarios
5. **05_scenario_geospatial.ipynb** - Large-scale geospatial scenario analysis

## HPC Access

These tutorials involve:
- Large-scale geospatial computations across multiple locations
- Distributed weather data retrieval
- Regional and global degradation mapping
- Geographic feature analysis (mountains, coastlines, rivers)
- Multi-dimensional spatial data processing

## Topics Covered

- Geospatial degradation mapping
- Geographic feature analysis and classification
- Regional climate-based modeling
- World-scale analysis and visualization
- KDTree-based spatial data structures
- Geospatial scenario workflows
